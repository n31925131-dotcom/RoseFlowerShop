# Οδηγός Χρήσης για την Άννα
## Πώς να Προσθέσετε Φωτογραφίες στην Ιστοσελίδα

---

## 📱 Καθημερινή Χρήση (Απλό!)

### Για να προσθέσετε φωτογραφίες:

1. **Ανοίξτε την εφαρμογή Dropbox** στο κινητό σας
2. **Πατήστε στον φάκελο:** `Rose-Flower-Shop-Gallery`
3. **Διαλέξτε τη σωστή κατηγορία:**
   - `weddings-γαμοι` → για γάμους
   - `baptisms-βαφτισια` → για βαφτίσεις
   - `events-εκδηλωσεις` → για εκδηλώσεις
   - `bouquets-μπουκετα` → για μπουκέτα
4. **Πατήστε το + (συν)** για να ανεβάσετε
5. **Επιλέξτε τις φωτογραφίες** από το κινητό σας
6. **Πατήστε Ανέβασμα**
7. **Τέλος!** Οι φωτογραφίες θα εμφανιστούν αυτόματα

---

## ⏰ Πόσο Καιρό Παίρνει;

- **Ανέβασμα φωτογραφιών**: 1-2 λεπτά
- **Εμφάνιση στην ιστοσελίδα**: Αυτόματα μέσα σε 30 λεπτά
- **Για άμεση εμφάνιση**: Πατήστε το κουμπί "Ανανέωση Γκαλερί" στην ιστοσελίδα

---

## 💡 Συμβουλές

### Για Ονόματα Φωτογραφιών

**Καλά παραδείγματα:**
```
γαμηλια-ανθοδεσμη-1.jpg
νυφικο-μπουκετο-τριανταφυλλα.jpg
βαφτιση-λευκα-λουλουδια.jpg
```

**Τι να αποφύγετε:**
```
IMG_1234.jpg  (όχι περιγραφικό)
γάμος 2024.jpg  (όχι κενά - χρησιμοποιήστε -)
```

### Πόσες Φωτογραφίες;

- ✅ Ανεβάστε 10-20 φωτογραφίες κάθε φορά
- ✅ Δεν υπάρχει όριο - ανεβάστε όσες θέλετε!
- ✅ Καλύτερα σε μικρές ομάδες παρά όλες μαζί

---

## 🔄 Για να Δείτε τις Φωτογραφίες Αμέσως

1. Πηγαίνετε στην ιστοσελίδα
2. Πατήστε "Γκαλερί"
3. Πατήστε το κουμπί "Ανανέωση Γκαλερί" (με το σύμβολο ↻)
4. Οι νέες φωτογραφίες θα εμφανιστούν!

---

## ❌ Για να Αφαιρέσετε Φωτογραφίες

### Επιλογή 1: Διαγραφή (Απλό)
1. Ανοίξτε Dropbox
2. Βρείτε τη φωτογραφία
3. Πατήστε παρατεταμένα → Διαγραφή
4. Θα εξαφανιστεί από την ιστοσελίδα σε 30 λεπτά

### Επιλογή 2: Αρχειοθέτηση (Προτεινόμενο)
1. Δημιουργήστε έναν φάκελο `_παλιες/` μέσα σε κάθε κατηγορία
2. Μετακινήστε τις παλιές φωτογραφίες εκεί
3. Δεν θα εμφανίζονται στην ιστοσελίδα αλλά θα παραμένουν αποθηκευμένες

---

## 🆘 Αν Κάτι Πάει Στραβά

### "Οι φωτογραφίες δεν εμφανίζονται"

**Ελέγξτε:**
1. ✓ Οι φωτογραφίες ανέβηκαν στο σωστό φάκελο;
2. ✓ Το ανέβασμα ολοκληρώθηκε; (δείτε στο Dropbox)
3. ✓ Περιμένατε 30 λεπτά ή πατήσατε "Ανανέωση";
4. ✓ Η φωτογραφία είναι .jpg ή .png;

### "Βλέπω λάθος φωτογραφίες"

- Ελέγξτε αν ανεβάσατε στη σωστή κατηγορία
- Μετακινήστε τη φωτογραφία στον σωστό φάκελο στο Dropbox

## 🎯 Παραδείγματα Χρήσης

### Παράδειγμα 1: Μετά από έναν Γάμο
```

2. Πηγαίνετε στο weddings-γαμοι
3. Ανεβάστε τις 15 καλύτερες φωτογραφίες
```
